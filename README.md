# speed-realestate-app

## 제작 이유❓
부모님이 부동산 매물 관리와 검색을 좀 더 쉽게 할 수 있도록 어플리케이션으로 제작

## ❇️  기능구현

## 1️⃣  메인페이지
### 메인페이지 상단 앱바
- [x] 부동산 이름 출력
- [x] 매물 등록 버튼생성

### 메인페이지 검색할 태그선택
- [x] 카테고리(아파트명, 거래유형, 타입, 동, 옵션)을 보여주는 컨텐츠 생성
- [x] 카테고리를 클릭하면 해당 카테고리의 태그를 선택 가능
- [x] 선택한 태그를 카테고리 밑에 출력

### 메인페이지 매물 검색
- [x] 바텀바를 이용해 매물검색 버튼 생성
- [x] 매물검색 버튼을 누를시 결과 페이지로 이동

## 2️⃣  매물 등록 페이지
### 매물 등록 페이지 상단 앱바
- [x] 부동산 이름 출력
- [x] 태그 추가 버튼생성

### 매물 등록 페이지 정보 입력
- [x] 아파트명, 타입, 동, 호, 거래유형, 가격, 옵션, 전화번호, 특이사항의 필드 생성
- [x] 태그와 문자열, 숫자를 이용해 입력하고 싶은 정보를 필드에 입력
- [x] 취소버튼 생성
- [x] 등록버튼 생성
- [x] 아파트명, 타입, 동, 호, 거래유형이 입력되지 않은경우 토스트 메세지로 메세지를 출력

### 매물 등록 페이지 태그 추가
- [x] 카테고리(아파트명, 타입, 동, 옵션)을 선택할 수 있는 드롭바 생성
- [x] 선택한 카테고리에 추가하고 싶은 태그를 입력할 수 있는 텍스트필드 생성
- [x] 텍스트필드에서는 ','를 사용하여 다수의 태그를 저장가능
- [x] 태그가 추가되었을 경우 토스트 메세지를 출력

### 매물 등록 취소
- [x] 취소버튼을 누를 경우 메인페이지로 이동

### 매물 등록 페이지 정보 등록
- [x] 등록버튼을 누를 경우 입력한 정보를 데이터베이스에 저장

## 3️⃣  매물 결과 페이지

### 매물 결과 페이지 앱바
- [x] 뒤로가기 버튼 생성
- [x] 부동산 이름 출력

### 매물 결과 페이지 뒤로가기
- [x] 뒤로가기 버튼을 누를경우 메인페이지로 이동

### 매물 결과 페이지
- [x] 검색된 매물의 갯수를 출력
- [x] 메인페이지에서 선택한 태그의 정보를 가지고 있는 매물을 데이터베이스로부터 출력
- [x] 선택된 태그가 없을 경우 모든 매물의 정보를 출력
- [x] 출력 정보는 아파트명, 타입, 동, 호, 거래유형, 가격으로 한정
- [x] 출력된 정보를 클릭하면 상세페이지로 이동

## 4️⃣  매물 상세 보기 페이지

### 매물 상세 보기 페이지
- [x] 매물 결과 페이지에서 선택한 매물의 모든 정보를 출력
- [x] 출력 정보는 아파트명, 타입, 동, 호, 거래유형, 가격, 옵션, 특이사항, 연락처
- [x] 전화번호를 클릭 시 바로 전화걸기 기능
- [x] 삭제버튼 생성
- [x] 수정버튼 생성
- [x] 닫기버튼 생성

### 매물 상세 보기 페이지 삭제
- [x] 매물 상세 보기 페이지에서 삭제버튼을 클릭할 경우 삭제를 확인하는 AlertDialog를 출력
- [x] AlertDialog에서 삭제를 누를경우 데이터베이스에서 해당 매물 삭제
- [x] AlertDialog에서 취소를 누를경우 AlertDialog 종료

### 매물 상세 보기 페이지 수정
- [x] 매물 상세 보기 페이지에서 수정버튼을 클릭할 경우 해당 정보가 적혀있는 매물 등록 페이지로 이동
- [x] 수정하기 버튼을 누를경우 해당 매물의 정보를 데이터베이스에서 업데이트
- [x] 취소버튼을 누를경우 매물 상세 보기 페이지로 이동

### 매물 상세 보기 페이지 닫기
- [x] 닫기버튼을 누를 경우 매물 결과 페이지로 이동
